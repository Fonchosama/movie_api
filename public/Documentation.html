REST API Documentation Site

This API gives you information about different movies, directors, and genres.

Bussines Logic URL HTTP Method Querry Parameters Requqest body data format Responde body data format
Return a list of ALL movies to the user /movies GET N/A N/A A JSON object with the data about all our movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/:title GET :name N/A A JSON object with the data of one movie
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genre/:genreName GET /movies/genre/:genreName N/A A JSON object with the description of one genre
Return data about a director (bio, birth year, death year) by name /movies/directors/:directorName GET :name N/A A JSON object with the data about a Director

Return data about one single user /users/:Username GET :Username N/A A JSON object with the data about a single user
Allow new users to register /users POST N/A A JSON object holding data about a new user (username, email, password) Successful POST request to register a new user
Allow users to update their user info (username) /users/:Username PUT :username A JSON object holding updated data about the username Successful PUT request to update the username
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) /users/:Username/movies/:MovieID PUT :MovieID A JSON object holding data about a favorite movie. Successful PUT request to add a movie to user favorites list
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) /users/:Username/movies/:MovieID DELETE :MovieID A JSON object holding data about a favorite movie to remove Successful DELETE request to delete a movie from the list of favorites
