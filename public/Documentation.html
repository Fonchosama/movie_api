REST API Documentation Site

This API gives you information about different movies, directors, and genres.

Bussines Logic URL HTTP Method Querry Parameters Requqest body data format Responde body data format
Return a list of ALL movies to the user /movies GET N/A N/A Successful GET request returning data of all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/:name GET :name N/A Successful GET request returning data of an specific movie
Return data about a genre (description) by name/title (e.g., “Thriller”) /genres/:name GET :name N/A Successful GET request returning data about a genre
Return data about a director (bio, birth year, death year) by name /directors/:name GET :name N/A Successful GET request returning data about a director
Allow new users to register /users/register POST N/A A JSON object holding data about a new user (username, email, password) Successful POST request to register a new user
Allow users to update their user info (username) /users/:username PUT :username A JSON object holding updated data about the username Successful PUT request to update the username
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) /users/:username/favorites PUT :username A JSON object holding data about a favorite movie. Successful PUT request to add a movie to user favorites list
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) /users/:username/favorites DELETE :username A JSON object holding data about a favorite movie to remove Successful DELETE request to delete a movie from the list of favorites
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) /users/:username DELETE :username N/A Successful DELETE request to delete an existing user
