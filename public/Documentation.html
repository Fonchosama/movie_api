REST API Documentation Site

This API gives you information about different movies, directors, and genres.

Bussines Logic URL HTTP Method Querry Parameters Requqest body data format Responde body data format
Return a list of ALL movies to the user /movies GET
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/:name GET
Return data about a genre (description) by name/title (e.g., “Thriller”) /genres/:name GET
Return data about a director (bio, birth year, death year) by name /directors/:name GET
Allow new users to register /users/register POST
Allow users to update their user info (username) /users/:username PUT
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) /users/:username/favorites PUT
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) /users/:username/favorites DELETE
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) /users/:username DELETE
