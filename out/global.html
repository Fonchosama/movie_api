PUT /users/:Username(req, res) → {Object}

This endpoint allows a user to update their information (username, email, password, and birthday). It first checks if the authenticated user is the same as the one whose data is being updated. If they are not the same user, it returns a "Permission denied" error. The password is hashed before being saved.
